This site was fully written, designed and coded by soda (me!).

I started learning html, css and js on my own, and originally made a website using github pages. As I started browsing little corners of the internet more and more, and finding small individual websites that functioned as little rooms to the people who created them, I realized that my previous website was quite stiff. So I briefly switched over to neocities, which actually helped me find my place and what I want to do.

This site is now my fourth revamp - I returned back to my original site UI, because the previous version was actually becoming much too gimmicky.
