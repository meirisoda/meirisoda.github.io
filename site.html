This site was fully written, designed and coded by soda (me!).

I started learning html, css and js on my own, and originally made a website using github pages. As i started browsing little corners of the internet more and more, and finding small individual websites that functioned as little rooms to the people who created them, i realized that my previous website was quite stiff. The repository for my github pages is still on my github profile, and you can visit if you want but I am far prouder of this website than I am of that one.

Rather, I am proud of the github pages website because I learned html/css/js through creating it, but I feel as though this website represents who I am as a person better, partly because I know what I want to recreate but also the very nature of neocities is for ordinary people to create and share small amateur websites.
For more information on my github pages, check out the interactive page on this site.



If you can't tell, I am very much inspired by early 2000s korean/japanese forums and blogs.
